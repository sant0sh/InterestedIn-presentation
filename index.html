Implementation Details

Node.js

  • Scalable

NoSQL Databases

  • Data is not stored in the form of tables and rows
  • Uses a BSON model which is more suited for network applications
  • Has support for handling Big Data and clustering
  • Scalable

Cloud implementation-Database and image processing

  • The mongo Database will reside on the cloud
  • The Advantage here is again scalability
  • The image processing too takes place over cloud
  • With these two important aspects of the project being on cloud its very easy to scale
